# 📌 Project Overview

The **AI-Powered Customer Support Ticketing System** is a Salesforce-based application developed as part of the **SmartBridge Internship Program** through **SkillWallet**.

The project automates customer support operations using **Salesforce Administration**, **Flow Automation**, and **Agentforce AI**.

It enables organizations to efficiently manage customer support requests by automating:

- Ticket Creation
- Case Classification
- Priority Assignment
- SLA Tracking
- Intelligent Agent Assistance
- Case Routing
- Reporting & Dashboards
- AI Feedback Collection

The solution demonstrates how Salesforce can be used to build a secure, scalable, and automation-driven customer support platform with minimal custom development.

---

# 🎯 Problem Statement

Organizations often face several challenges while managing customer support:

- Manual ticket classification
- Delayed response times
- Poor SLA tracking
- Limited visibility into support performance
- Lack of intelligent assistance for agents
- Inconsistent customer service processes

A centralized and automated support management system was required to improve productivity, reduce manual effort, and ensure timely issue resolution.

## 🚦 Priority Assignment

Automatically assigns:

- 🟠 High

##  SLA Management

Automatically:

- Tracks First Response Time
- Monitors Resolution Deadlines
- Supports Escalation Process

##  Agentforce AI Assistance

Provides intelligent recommendations including:

- Suggested Actions
- Similar Case Recommendations
- AI-powered Guidance
- Faster Issue Resolution

##  AI Feedback Loop
